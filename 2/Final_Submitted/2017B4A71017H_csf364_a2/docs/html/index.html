Segmented Least Squares Algorithm
Brief description

Overview

This project implements the Segmented Least Squares Algorithm to solve the line fitting problem using dynamic programming. It finds multiple best fit lines in the optimal partition.

Input and running

Compile by using the command g++ -std=c++17 sls.cpp main.cpp -o k
Run by using the command k < "..\input\in.txt"
Visualize by using the command python visualization.py
The input is stored in input folder, must be .txt file and must represent a set of points (x, y) in the 2-D plane
Sample input has been included in input/
All output will be generated in output/
