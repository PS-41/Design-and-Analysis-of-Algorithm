int main ( )

Generates test cases to be used by the algorithms Takes the number of points n, cost value and required number of segments as input from the user. Writes the generated test case in the in.txt file. n random points are taken on the x-axis. We take first n / segments points and lift them by y = mx + c where m and c are random values for this set of points. Similarly, we lift the other sets of points.
